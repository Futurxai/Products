# EatWell4U v1.0.0 — Release Notes

**Release Date:** 2025-04-05
**APK Size:** 7.2 MB
**Min Android:** 8.0 (API 26)

## What's New
- Initial release
- Google Sign-In authentication
- 6-step profile setup wizard
- Daily meal dashboard with calorie tracking
- 50+ recipes across 5 cuisines (South Indian, North Indian, Continental, Chinese, Healthy)
- Weekly 7-day meal planner
- Grocery list with category filters
- Dark mode support
- Push notification meal reminders
- Camera food scanner with healthier swap suggestions

## Known Issues
- Camera scanner requires good lighting for accurate detection
- PDF export may take a few seconds on older devices
