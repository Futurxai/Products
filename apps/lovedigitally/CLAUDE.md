# LoveDigitally — App Context

What it does: Digital relationship and social connection platform
Target user: Young adults in India
Key features: User profiles, messaging, event coordination, connection matching
Shared packages: shared-auth, shared-notifications, shared-analytics, shared-supabase
Current phase: Phase 1 — Base Setup

App-specific rules:
- Privacy first — no user data visible between profiles without mutual consent
- All messages encrypted at rest in Supabase
- Notification for every new connection request
