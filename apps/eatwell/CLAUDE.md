# EatWell — App Context

What it does: Nutrition and meal planning app for Indian users
Target user: Health-conscious individuals in Tamil Nadu
Key features: Meal tracking, local Indian food database, calorie counting, diet plans
Shared packages: shared-auth, shared-analytics, shared-supabase
Current phase: Phase 1 — Base Setup

App-specific rules:
- Use local Indian food names and nutritional data
- Support Tamil and English language
- All food data stored in Supabase meals table
